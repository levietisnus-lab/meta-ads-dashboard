⚙️ Token Nhắn tin
Cấu hình Page Access Token có quyền pages_messaging cho từng trang
Cách lấy Page Access Token:
1. Vào Meta Business Suite → Cài đặt → Tài khoản → Trang → chọn trang → Thông tin
2. Hoặc dùng Graph API Explorer: chọn App → chọn Page → Generate Token → tích pages_messaging
3. Token phải là Page Access Token (không phải User Token), không hết hạn nếu chuyển sang Long-lived
